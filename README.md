# Financial Literacy

This website is designed to help users learn about financial literacy. It includes information on what financial literacy is, why it's important, and resources for further learning. The website also has a quiz that users can take to test their knowledge of financial literacy.

## Technologies Used

The website is built using HTML, CSS, and JavaScript. The front end is built using Bootstrap, and the back end is built using Flask. The website also uses the Owl Carousel library for the slideshow.
